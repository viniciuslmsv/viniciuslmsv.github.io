Exemplos de navbar

Este exemplo é um simples exercício para ilustrar como a navbar e seus conteúdos funcionam. Algumas navbars se extendem até a largura da viewport, enquanto outras são presas a largura de um .container. Para saber sobre posicionamento, veja os exemplos de navbar no topo e fixa ao topo.

No menor breakpoint, o plugin de colapso é ativado, esconde os links e mostra um menu hambúrguer para alternar a visibilidade do conteúdo oculto.
